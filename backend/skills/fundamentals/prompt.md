# Fundamentals Skill Guidelines

## When to Use
- User asks about company valuation (P/E, P/B, PEG, etc.)
- User wants to know profitability metrics (ROE, margins, etc.)
- User inquires about financial health (debt, cash, ratios)
- User asks about earnings history or financial reports

## Tool Selection
- `get_company_fundamentals`: For valuation, profitability, financial health metrics
- `get_earnings_history`: For quarterly/annual earnings trends

## Best Practices
- Explain metrics in context (e.g., "P/E of 25 is high compared to industry average")
- Compare metrics across time periods when available
- Mention data source and update frequency
- Always include disclaimer that data is for reference only

## Important Notes
- Fundamental data is typically updated quarterly
- Some metrics may not be available for all companies
- Always verify ticker symbol is correct before analysis
