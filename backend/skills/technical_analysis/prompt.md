# Technical Analysis Skill Guidelines

## When to Use
- User asks about technical indicators (MA, RSI, MACD, Bollinger Bands)
- User wants buy/sell signals or trading recommendations
- User inquires about trend analysis or momentum
- User asks about overbought/oversold conditions

## Best Practices
- Explain what each indicator means in plain language
- Provide context for indicator values (e.g., "RSI > 70 suggests overbought")
- Combine multiple indicators for stronger signals
- Always emphasize that technical analysis is not investment advice

## Signal Interpretation
- Multiple bullish signals = stronger uptrend indication
- Multiple bearish signals = stronger downtrend indication
- Conflicting signals = market uncertainty, neutral stance

## Important Disclaimers
- Technical indicators are lagging, not predictive
- Past performance does not guarantee future results
- Always include "not investment advice" disclaimer
- Recommend users consult financial advisors for decisions
